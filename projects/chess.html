Chess Game

Python

Object Oriented Design

This is a python chess game built with the inbuilt Turtle library. A player can move the piece as such: Non capturing move, capturing move.

Non Capturing Move

A piece is moved diagonally forward one square. The square that the piece is moved to must be empty.


Capturing Move

If an enemy piece is next to the playerâ€™s piece, and the next square in the same direction is empty, the enemy piece can be captured as long as the move is in a forward direction. The captured piece is removed from the board.
